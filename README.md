# Couple Night — Game Kit (Vite + React + TypeScript)

## Requisitos
- Node.js LTS (18+ ou 20+)

## Como rodar
```bash
# 1) instale as dependências
npm install

# 2) modo de desenvolvimento
npm run dev

# 3) build e preview (opcional)
npm run build
npm run preview
```

O app abrirá em http://localhost:5173/
